这是一个大长URL。您的浏览器并不能很好地与Kibana的bacon-double-cheese-burger-with-extra-fries sized URL工作。 为了避免遇到问题，Kibana将浏览器中的URL限制为{{controller.limit}}个字符，以确保您的安全。

如何解决这个问题?

这通常只发生在大型，复杂的仪表板上，所以你有一些选择:

  1. 在高级设置中启用state：store SessionStorage 选项。 这样可以防止这些URL变得越来越长，但是它们的便携性却不太可观.
  2. 从仪表板中移除一些东西。 这将减少URL的长度，并保持IE在一个好的地方.
  3. 不要使用IE。 我们知道的每个其他支持的浏览器都没有这个限制.
