脚本帮助

默认 Elasticsearch脚本使用Lucene表达式 ：这是一个很像JavaScript，但仅限于基本的算术运算，运算和比较运算。我们会让你在Lucene表达式上做一些阅读来访问文档中的值使用下面的格式：

doc['some_field'].value

使用Lucene表达式的一些限制:

  • 数值字段
  • 存储(Stored)字段不可用
  • 如果字段稀疏（只有一些文档包含值），缺少字段的文档将有0的值

下面是脚本字段中可用的操作:

  • 算术运算符: + - * / %
  • 位运算符: | & ^ ~ << >> >>>
  • 布尔运算符 (包含三元运算符): && || ! ?:
  • 比较运算符: < <= == >= >
  • 通用的数学函数: abs ceil exp floor ln log10 logn max min sqrt pow
  • 三角函数库: acosh acos asinh asin atanh atan atan2 cosh cos sinh sin tanh tan
  • 距离函数: haversin
  • 其他功能: min, max
